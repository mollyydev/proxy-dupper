# Proxy Dupper

Язык: **Русский** | [English](README.md)

Proxy Dupper берёт список прокси с авторизацией, группирует их по /24 сети, генерирует все IP в каждой /24, проверяет и сохраняет валидные.

## Требования
- Python 3.10+
- Доступ в интернет (для проверки прокси)

## Установка (uv)
```bash
uv venv
uv sync
```

## Установка (pip)
```bash
python -m venv .venv
.venv\Scripts\Activate.ps1
pip install -e .
```

## Подготовка входного файла
Создайте `proxies.txt`, по одному прокси на строку:
```
login:password@1.2.3.4:8080
```

## Запуск
```bash
python main.py -i proxies.txt -o valid.txt -p http -w 20 -t 8 --test-url http://httpbin.org/ip
```

## Результат
Валидные прокси записываются в файл, указанный через `-o`.

## Параметры CLI
- `-i, --input` Путь к файлу с прокси (обязательный)
- `-o, --output` Файл для валидных прокси (обязательный)
- `-p, --protocol` `http` или `socks5` (по умолчанию: `http`)
- `-w, --workers` Лимит параллельных задач (по умолчанию: `10`)
- `-t, --timeout` Таймаут запроса в секундах (по умолчанию: `8`)
- `--test-url` URL для проверки прокси (по умолчанию: `http://httpbin.org/ip`)

## Примечания
- Если в исходном файле несколько прокси из одной /24, используется только первый.
- Проверка выполняется запросом к `http://httpbin.org/ip`.
